Acerca del Proyecto

Weather Frontend M2 es un proyecto académico que consiste en desarrollar un MVP (Minimum Viable Product) de una aplicación de clima utilizando HTML5 semántico, Bootstrap para estilos y diseño responsivo, y JavaScript básico para la interacción.

Temática

La aplicación muestra el clima de ciudades icónicas alrededor del mundo, como París, Nueva York, Tokio, Londres, Roma, Río de Janeiro, El Cairo, Sídney, Ciudad de México y Santiago.

Objetivos de aprendizaje

  • Practicar la estructuración de páginas con HTML5 semántico.
  • Aplicar Bootstrap para un layout responsivo (mobile-first).
  • Manipular el DOM con JavaScript (eventos y estados simples).
  • Gestionar el proyecto con Git/GitHub.
